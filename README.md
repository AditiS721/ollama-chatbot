\# 🤖 Local AI Chatbot with Ollama



A private, local AI chatbot built with Ollama and Gradio. Runs completely offline on your machine.



\## Features

\- 💬 Text chat with AI

\- 🖼️ Image understanding (multimodal)

\- 📁 File upload support

\- 🔒 100% Private - all processing done locally

\- 🎨 Clean web interface



\## Prerequisites

\- Python 3.10 or higher

\- \[Ollama](https://ollama.ai) installed

\- At least 8GB RAM recommended
